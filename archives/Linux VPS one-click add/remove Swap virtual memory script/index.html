Linux VPS 一键添加/删除 Swap 虚拟内存脚本

Moexin 2020-03-10 in 脚本 · Views

很多人的VPS服务器由于内存太小，会导致很多进程被杀掉，这时候就需要我们添加Swap虚拟内存了，这里就整了个一键脚本方便懒人或小白使用。

使用方法

注意： 脚本不支持OpenVZ架构，安装会自动退出。

bash <(wget --no-check-certificate -qO- https://cdn.jsdelivr.net/gh/Moexin/Shell@master/Swap.sh)

然后根据选项进行操作，记得添加Swap的时候填写纯数字，默认单位为M。
